## Overview

Led the development of predictive models at IT Shoulders Inc, leveraging large-scale datasets and multivariate analysis techniques to forecast user behaviors. The project involved end-to-end data science workflows from data collection and feature engineering through model training, evaluation, and deployment.

## Key Features

- **Feature Engineering**: Extracted and engineered features from complex, multi-dimensional datasets to improve model performance.
- **Model Development**: Built and compared multiple machine learning models including Random Forests, XGBoost, and neural networks for behavior prediction.
- **Dashboard Analytics**: Created interactive dashboards to visualize prediction results and uncover trends in customer behavior.
- **Statistical Analysis**: Applied advanced statistical methods to uncover patterns and validate model assumptions.

## Technologies Used

- **Python**: pandas, scikit-learn, XGBoost, matplotlib, seaborn
- **SQL**: Complex queries for data extraction and aggregation
- **Tableau**: Interactive visualization and reporting dashboards
- **AWS**: Cloud infrastructure for model training and deployment

## Impact

The predictive models achieved high accuracy in forecasting user behaviors, enabling the business team to make data-driven decisions about product features, marketing campaigns, and customer retention strategies.
