## Overview

Developed a comprehensive AI chatbot comparison platform enabling side-by-side evaluation of OpenAI GPT-4o and Google Gemini Flash models. The platform includes real-time token usage tracking, cost optimization analytics, and response quality comparison features.

## Key Features

- **Side-by-Side Comparison**: Real-time parallel execution and display of responses from multiple AI models.
- **Token Usage Tracking**: Granular monitoring of input and output tokens with cost calculations per request and session.
- **Cost Optimization**: Analytics dashboard showing cost-per-query comparisons to help teams choose the most cost-effective model.
- **React Frontend**: Clean, responsive interface built with React for seamless user interaction.

## Technologies Used

- **React**: Frontend framework for the comparison interface.
- **FastAPI**: Backend API handling model orchestration.
- **OpenAI API**: GPT-4o integration for response generation.
- **Google Gemini API**: Gemini Flash model integration.
- **Python**: Backend logic, API orchestration, and analytics.

## Impact

The platform enabled data-driven decisions about which AI model to deploy for specific use cases, resulting in significant cost savings while maintaining response quality standards.
