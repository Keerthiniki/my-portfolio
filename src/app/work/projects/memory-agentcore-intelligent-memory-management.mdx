## Overview

Memory AgentCore is an intelligent memory management system designed for AI agents, featuring a dual-tier memory architecture (short-term and long-term memory) built on FastAPI. The system enables AI agents to maintain context across conversations and sessions, dramatically improving the quality of AI interactions.

## Key Features

- **Dual-Tier Memory Architecture**: Implemented separate short-term memory (STM) and long-term memory (LTM) systems with intelligent overflow handling between tiers.
- **PostgreSQL Adapter**: Built a robust database layer with PostgreSQL for persistent memory storage with encryption key management.
- **Authentication & Security**: Designed comprehensive authentication methods and encryption protocols for secure memory access.
- **Multi-Agent Support**: Architected the system to support multiple AI agents sharing and accessing memory pools.

## Technologies Used

- **FastAPI**: High-performance async Python web framework for the API layer.
- **PostgreSQL**: Relational database for persistent memory storage.
- **AWS**: Cloud deployment infrastructure including Lambda and S3.
- **Docker**: Containerized deployment for consistent environments.
- **Python**: Core development with async/await patterns.

## Challenges and Learnings

The project involved addressing 17 technical gaps identified during code review, spanning authentication, deployment, security, STM overflow handling, and encryption key management. This iterative improvement process strengthened the overall system architecture significantly.
